# Code of Conduct

We are committed to keeping this project welcoming, respectful, and inclusive for everyone.

## Expectations

- Be kind, respectful, and constructive
- Use inclusive and professional language
- Respect differing opinions and experiences

## Unacceptable Behavior

- Harassment, discrimination, or abusive behavior
- Personal attacks, trolling, or sharing private information without consent

## Enforcement

Project maintainers may take appropriate action if these guidelines are violated.

By participating in this project, you agree to follow this Code of Conduct.
